HIERBAS

Una hierba es una planta de tamaño pequeño que presenta un tallo tierno y no leñoso.

La hierba que cubre el terreno se conoce como césped.

Aquella que se emplea para que el ganado se alimente en el lugar donde se cría, se conoce como pasto.

También se le dice hierba a las plantas con propiedades medicinales o que se utilizan en la gastronomía.

Las hierbas medicinales,son vegetales que se usan para el tratamiento de trastornos de salud y enfermedades.

Entre las plantas medicinales están:

  • El romero

  • La manzanilla

  • La menta

  • La salvia

  • La hierba de San Juan

    Las hierbas aromáticas, son las plantas que se emplean en la cocina como condimento, tanto frescas como secas, como el:

  • Perejil

  • El tomillo

  • El orégano
